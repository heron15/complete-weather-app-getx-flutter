# Modern Weather App.

Modern weather app manage with GetX state management.

# Features

<ol>
    <li>
        Using GetX state management.
    </li>
    <li>
        Background animation changes based on current weather.
    </li>
    <li>
        Using Geolocator for user current location auto detect.
    </li>
</ol>
